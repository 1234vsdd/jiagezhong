媒体关系建立与维护指南

加个钟 | 原创内容

为什么要建立媒体关系？

在信息时代，媒体关系是品牌传播的核心资产。

媒体关系的价值：

  • 新闻报道比广告更具可信度
  • 媒体是危机时的传声筒
  • 好的媒体关系 = 低成本高回报
  • 媒体资源 = 品牌话语权

    记者关系维护

    1. 了解记者需求

  • 他们的选题偏好是什么？
  • 他们的截稿时间是什么时候？
  • 他们需要什么样的素材？
  • 他们的痛点是什么？

    • 2. 提供有价值的信息

  • 独家新闻线索
  • 行业数据报告
  • 专家观点洞察
  • 案例素材支持

    • 3. 建立长期信任

  • 及时响应媒体询问
  • 信守承诺（说好的资料一定要给）
  • 建立个人关系（节日问候、线下交流）
  • 保持沟通连续性

    • 新闻稿发布技巧

    1. 什么值得发新闻稿？

  • 公司重大战略发布
  • 新产品/服务上线
  • 融资/上市消息
  • 战略合作签约
  • 重要活动/事件
  • 社会公益活动
  • 获奖/认证消息

    • 2. 什么不值得发？

  • 日常促销活动
  • 公司内部动态
  • 价值不大的"好消息"
  • 过于常规的内容

    • 3. 新闻稿结构

    标准结构：

    1. 标题（简洁有力、包含亮点）

    2. 副标题（补充说明）

    3. 导语（最重要的事实，5W1H）

    4. 主体（详细叙述、背景支撑）

    5. 引用（高管/专家观点）

    6. 关于我们（公司简介）

    7. 联系方式（媒体联络人）

    4. 发布渠道

    媒体活动策划

    1. 新闻发布会

    适用场景： 重大战略发布、融资公布等

    关键要素：

  • 明确的新闻点
  • 合适的时机
  • 核心媒体邀请
  • 高管出席
  • Q&A准备

    • 2. 媒体沟通会

    适用场景： 深度交流、专项话题

    关键要素：

  • 精选媒体名单
  • 深度话题准备
  • 参观/体验环节
  • 后续跟进计划

    • 3. 媒体参观/体验

    适用场景： 产品发布、品牌展示

    关键要素：

  • 精心设计的动线
  • 产品演示环节
  • 体验/试用安排
  • 素材拍摄支持

    • 4. 线上媒体活动

    适用场景： 远程沟通、成本控制

    关键要素：

  • 稳定的直播平台
  • 提前测试设备
  • 互动环节设计
  • 后续内容沉淀

    • 危机时期的媒体沟通

    危机时媒体关系的原则

    1. 快速响应 - 第一时间联系核心媒体

    2. 统一口径 - 所有对外发言一致

    3. 坦诚沟通 - 不隐瞒、不撒谎

    4. 持续更新 - 保持信息畅通

    危机时避免的错误

  • ❌ 拒绝沟通、逃避采访
  • ❌ 推卸责任、指手画脚
  • ❌ 前后矛盾、说法不一
  • ❌ 暗中操作、试图"摆平"

    • 记者沟通话术

    接受采访时的技巧

  • 核心信息提前准备（不超过3点）
  • 用数据和案例支撑观点
  • 避免"无可奉告"
  • 不会回答的问题可以转移话题
  • 重要观点可以要求"不具名引用"

    • 拒绝采访的技巧

  • 真诚说明原因
  • 提供替代方案（书面声明、安排其他受访人）
  • 保持礼貌和尊重
  • 记录并后续跟进
